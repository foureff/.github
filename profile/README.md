# 4F – Website Failure Detection & Monitoring

**4F** is a broken link checker and website monitoring service available at [4f.at](https://www.4f.at).

It crawls your site and reports:

- Broken links (4xx/5xx errors)
- Missing assets (images, scripts, stylesheets)
- Broken anchor targets
- Redirect chains
- Slow responses
- SSL/TLS errors
- Mixed content

No account needed for a one-off scan. Free and paid plans are available for scheduled monitoring, multiple projects, team access, and API/webhook integrations.
